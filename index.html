Android Plugin DSL Reference

This is the home of Android Gradle Plugin DSL reference. For instructions on how to work on the plugin itself, please see Building the Android Gradle Plugin. To file bugs against the plugin or Android Studio, please go to b.android.com and choose the right template ("Android Studio Bug" or "Tools bug report").

Please choose the plugin version:

  • Current (1.5)
  • 1.5
  • 1.4
  • 1.3
  • 1.2

If you want to write a new plugin that takes advantage of the Android public APIs, see the JavaDoc page.
